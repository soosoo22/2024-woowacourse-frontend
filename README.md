## 🏋️ 레벨 1

###  학습 목표

레벨1의 목표는 `프로그래밍 기본` 역량을 탄탄하게 쌓는 것이다.

- 소규모 애플리케이션 개발을 통해 JavaScript/TypeScript의 주요 문법을 깊이 학습한다.
- 유지보수성 좋은 코드의 필요성을 체감하고, 이를 작성하는 방법을 고민하며 적용한다.
